# DriveIndex

> 此项目正在开发中，暂无路线图以及时间表。

DiveIndex 后端的 SpringBoot 实现。

## 使用

使用以下命令启动应用：

```shell
java -jar driveindex.jar
```

可附加参数指定配置文件：

```shell
java -jar driveindex.jar --config="./config/driveindex.ini"
```

## 开发环境

+ IDE：JetBrains IDEA Ultimate 2025.2
+ Java：JDK 21
+ Database：PostgreSQL 16
